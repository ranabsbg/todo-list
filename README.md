# todo-list
# To-Do List Uygulaması

PHP ve MySQL kullanılarak geliştirilmiş bir görev yönetim sistemidir.

## Özellikler
- Görev ekleme
- Görev silme
- Görev tamamlama
- Kategori sistemi
- Arama ve filtreleme
- Öncelik sistemi

## Kullanılan Teknolojiler
- PHP
- MySQL
- HTML
- CSS

## Amaç
Bu proje CRUD işlemlerini öğrenmek ve temel web uygulaması geliştirmeyi amaçlamaktadır.
